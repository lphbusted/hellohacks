Challenge 1: Is the Number Even or Odd?

Read the code below. If the number is even, the program will print "Even". If it's odd, it will print "Odd".


let num = 7;

if (num % 2 === 0) {
    console.log("Even");
} else {
    console.log("Odd");
}
